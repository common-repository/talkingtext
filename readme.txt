== Description ==

Free plugin to convert blog entries to speech (text-to-speech technology). Blog entries will be converted into audio files. A flash player to playback audio files is provided. Information on how to activate your plugin can be found here: <a href="http://www.talkingtext.com">talkingtext.com</a>.
